Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* androidLibs/maps.jar => app/androidLibs/maps.jar
* libs/android-support-v4.jar => app/libs/android-support-v4.jar
* libs/dom4j-1.6.1.jar => app/libs/dom4j-1.6.1.jar
* libs/jaxen-1.1.1.jar => app/libs/jaxen-1.1.1.jar
* libs/junit-4.10.jar => app/libs/junit-4.10.jar
* libs/libimc.jar => app/libs/libimc.jar
* src/ => app/src/main/java/
* test/ => app/src/main/java/
